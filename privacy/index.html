2. Charakter aplikace

SnapFaX je mobilní aplikace pro iOS, která slouží k rozpoznávání a identifikaci objektů (zejména památek, budov a zajímavých míst) pomocí kamery zařízení a automatizované analýzy obrazu prostřednictvím umělé inteligence.

Aplikace nevyžaduje registraci, vytvoření uživatelského účtu ani přihlášení.

3. Jaké údaje jsou zpracovávány

3.1 Obrazová data

  • uživatel může pořídit fotografii nebo nahrát obrázek z galerie zařízení,
  • obrazová data jsou jednorázově odeslána k analýze pomocí umělé inteligence,
  • Správce obrazová data dlouhodobě neukládá,
  • po dokončení analýzy jsou obrazová data odstraněna.

Do lokální historie zařízení se ukládají pouze textové výsledky rozpoznání, nikoli samotné fotografie.

3.2 Poloha zařízení

  • Aplikace pracuje s přesnou polohou (GPS),
  • poloha je využívána výhradně lokálně v zařízení uživatele,
  • slouží k filtrování obsahu podle vzdálenosti a zobrazení na mapě,
  • poloha není odesílána na server ani třetím stranám,
  • přístup k poloze je zcela volitelný.

3.3 Technické údaje

  • typ zařízení a verze operačního systému,
  • IP adresa v rámci síťové komunikace,
  • základní provozní a technická metadata.

4. Právní základ zpracování

  • zpracování obrazových dat – plnění smlouvy (poskytnutí funkce Aplikace),
  • zpracování polohy – souhlas uživatele,
  • technické údaje – oprávněný zájem na provozu a zabezpečení Aplikace.

5. Umělá inteligence a třetí strany

Aplikace využívá služby třetích stran, zejména:

  • Google Gemini Vision API – analýza obrazu a generování textu,
  • Cloudflare Workers – technická proxy,
  • Supabase – backendová infrastruktura.

Do systému umělé inteligence je odesílán pouze obraz a textový prompt. Informace o poloze nejsou součástí AI dotazů.

Data mohou být přenášena mimo Evropskou unii (zejména do USA). Přenos probíhá na základě standardních smluvních doložek (SCC) dle čl. 46 GDPR.

6. Reklamy

  • Aplikace zobrazuje odměňované reklamy,
  • reklamy jsou poskytovány prostřednictvím služby Google AdMob,
  • Aplikace nevyužívá App Tracking Transparency (IDFA),
  • reklamy jsou zobrazovány jako nepersonalizované.

7. Doba uchování údajů

  • obrazová data – pouze po dobu nezbytnou k provedení analýzy,
  • historie skenů – lokálně v zařízení uživatele,
  • Správce nevede centrální databázi uživatelských profilů.

8. Práva uživatele

  • právo na informace o zpracování údajů,
  • právo na přístup k údajům,
  • právo na výmaz údajů,
  • právo vznést námitku,
  • právo podat stížnost u ÚOOÚ.

9. Ochrana dětí

Aplikace není určena pro děti mladší 13 let.

10. Změny zásad

Správce si vyhrazuje právo tyto zásady kdykoli aktualizovat. Aktuální verze je vždy veřejně dostupná.

Privacy Policy of the SnapFaX Application

Effective from: December 24, 2025

This Privacy Policy explains how the SnapFaX mobile application (the “Application”) processes personal data and related information when used.

1. Data Controller

Filip Šťastný
natural person – non-entrepreneur
Prague, Czech Republic

(the “Controller”)

2. Nature of the Application

SnapFaX is an iOS mobile application designed to recognize and identify objects, particularly landmarks, buildings, and points of interest, using the device camera and automated image analysis powered by artificial intelligence.

The Application does not require registration or the creation of a user account.

3. Data Processed

3.1 Image Data

  • users may capture photos or upload images from their device gallery,
  • image data is sent once for analysis using artificial intelligence,
  • the Controller does not store image data long-term,
  • image data is deleted after analysis is completed.

Only text-based recognition results are stored locally on the user’s device.

3.2 Device Location

  • the Application uses precise GPS location,
  • location data is processed exclusively locally on the device,
  • it is used to filter content by distance and display maps,
  • location data is not transmitted to servers or third parties,
  • granting location access is entirely optional.

3.3 Technical Data

  • device type and operating system version,
  • IP address within network communication,
  • basic operational and technical metadata.

4. Legal Basis for Processing

  • image data processing – performance of a contract,
  • location processing – user consent,
  • technical data – legitimate interest in operating and securing the Application.

5. Artificial Intelligence and Third Parties

The Application uses third-party services, in particular:

  • Google Gemini Vision API – image analysis and text generation,
  • Cloudflare Workers – technical proxy services,
  • Supabase – backend infrastructure.

Only image data and a text prompt are transmitted to the AI system. Location data is not included in AI requests.

Data may be transferred outside the European Union (in particular to the United States). Such transfers are carried out based on Standard Contractual Clauses (SCC) pursuant to Article 46 of the GDPR.

6. Advertisements

  • the Application displays rewarded advertisements,
  • ads are provided via Google AdMob,
  • the Application does not use App Tracking Transparency (IDFA),
  • ads are effectively displayed as non-personalized.

7. Data Retention

  • image data – retained only for the time necessary to perform analysis,
  • scan history – stored locally on the user’s device,
  • no central user profile database is maintained.

8. User Rights

  • right to information about data processing,
  • right of access to personal data,
  • right to erasure of data,
  • right to object to processing,
  • right to lodge a complaint with a supervisory authority.

9. Protection of Children

The Application is not intended for children under the age of 13.

10. Changes to This Policy

The Controller reserves the right to update this Privacy Policy at any time. The current version will always be publicly available.
