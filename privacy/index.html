2. O jakou aplikaci se jedná

SnapFaX je mobilní aplikace pro iOS, která funguje jako průvodce v rozšířené realitě. Pomocí kamery rozpoznává památky a další objekty a zobrazuje jejich historický nebo informační kontext. Aplikace využívá umělou inteligenci a veřejně dostupné zdroje dat.

Používání aplikace nevyžaduje registraci ani vytvoření uživatelského účtu.

3. Jaké údaje aplikace zpracovává

3.1 Fotky a obrazová data

Aplikace umožňuje pořizovat snímky pomocí kamery nebo nahrát obrázek z galerie zařízení za účelem jejich analýzy.

  • snímek je zpracován za účelem rozpoznání objektu,
  • snímek může být dočasně odeslán na vzdálený server k analýze pomocí umělé inteligence,
  • snímky nejsou Správcem dlouhodobě ukládány na serveru.

Historie úspěšných skenů je ukládána pouze lokálně v zařízení uživatele a může být kdykoli smazána.

3.2 Poloha zařízení

Aplikace může pracovat s přibližnou polohou zařízení, pokud k tomu uživatel udělí oprávnění.

  • přístup k poloze je zcela volitelný,
  • poloha je používána pouze při aktivním používání aplikace,
  • slouží k doporučení obsahu v okolí uživatele,
  • poloha není sledována na pozadí ani dlouhodobě ukládána.

3.3 Technické údaje

Při používání aplikace mohou být technicky zpracovávány běžné provozní údaje, například:

  • typ zařízení,
  • verze operačního systému,
  • IP adresa (v rámci síťové komunikace).

Tyto údaje jsou zpracovávány pouze v rozsahu nutném pro fungování aplikace a komunikaci se servery třetích stran.

4. Umělá inteligence a třetí strany

Aplikace využívá služby třetích stran k analýze obrazu a generování textových informací, zejména:

  • Cloudflare – jako technickou proxy,
  • Google (Gemini) – pro analýzu obrazu a generování popisů.

Obrazová data a související technické informace mohou být v rámci tohoto procesu předána mimo Evropskou unii, zejména do Spojených států amerických.

Přenos probíhá pouze za účelem poskytnutí funkčnosti aplikace a v souladu s platnými právními předpisy o ochraně osobních údajů.

5. Reklamy

Aplikace zobrazuje rewarded reklamy (odměňované reklamy), které umožňují uživateli získat dodatečné funkce nebo bonusy v aplikaci.

  • reklamy jsou poskytovány prostřednictvím služby Google AdMob,
  • reklamy jsou zobrazovány v režimu nepersonalizovaných reklam,
  • aplikace nevyužívá osobní údaje uživatele k cílení reklam.

Třetí strany mohou v rámci zobrazování reklam zpracovávat omezené technické údaje v souladu se svými zásadami ochrany soukromí.

6. Jak dlouho jsou údaje uchovávány

  • obrazová data jsou uchovávána pouze po dobu nezbytnou k analýze,
  • historie skenů je uložena lokálně v zařízení uživatele,
  • žádná centrální databáze uživatelských profilů není vedena.

7. Práva uživatele

Uživatel má v souladu s GDPR zejména právo:

  • na informace o zpracování osobních údajů,
  • na přístup ke svým údajům,
  • na výmaz údajů (v rozsahu, v jakém jsou zpracovávány),
  • vznést námitku proti zpracování,
  • podat stížnost u dozorového úřadu (ÚOOÚ).

8. Ochrana dětí

Aplikace není určena pro děti mladší 13 let a necílí na děti jako zvláštní skupinu uživatelů.

9. Změny těchto zásad

Správce si vyhrazuje právo tyto zásady ochrany soukromí aktualizovat, zejména v případě změn funkčnosti aplikace nebo právních předpisů. Aktuální verze bude vždy dostupná veřejně.

10. Kontakt

V případě dotazů ohledně ochrany soukromí se může uživatel obrátit na Správce prostřednictvím e-mailu uvedeného výše.

Privacy Policy of the SnapFaX Application

Effective from: December 13, 2025

These privacy policy explain how the SnapFaX application (hereinafter referred to as the “Application”) processes personal data and other information when it is used.

The purpose is to provide clear and transparent information about what data is processed, for what purpose, and what rights the user has.

2. Description of the Application

SnapFaX is an iOS mobile application that functions as an augmented reality guide. Using the camera, it recognizes landmarks and other objects and displays their historical or informational context. The Application uses artificial intelligence and publicly available data sources.

Using the Application does not require registration or the creation of a user account.

3. What Data the Application Processes

3.1 Photos and Image Data

The Application allows capturing images using the camera or uploading an image from the device gallery for analysis purposes.

  • the image is processed for the purpose of object recognition,
  • the image may be temporarily sent to a remote server for analysis using artificial intelligence,
  • images are not stored long-term on the server by the Controller.

The history of successful scans is stored only locally on the user’s device and can be deleted at any time.

3.2 Device Location

The Application may work with the approximate location of the device if the user grants permission.

  • access to location is entirely optional,
  • location is used only during active use of the Application,
  • it serves to recommend content near the user,
  • location is not tracked in the background or stored long-term.

3.3 Technical Data

When using the Application, common technical operational data may be processed, for example:

  • device type,
  • operating system version,
  • IP address (within network communication).

This data is processed only to the extent necessary for the operation of the Application and communication with third-party servers.

4. Artificial Intelligence and Third Parties

The Application uses third-party services for image analysis and generation of textual information, in particular:

  • Cloudflare – as a technical proxy,
  • Google (Gemini) – for image analysis and description generation.

Image data and related technical information may, within this process, be transferred outside the European Union, in particular to the United States of America.

The transfer takes place solely for the purpose of providing the Application’s functionality and in accordance with applicable personal data protection laws.

5. Advertisements

The Application displays rewarded advertisements, which allow the user to obtain additional features or bonuses in the Application.

  • advertisements are provided through the Google AdMob service,
  • advertisements are displayed in non-personalized advertising mode,
  • the Application does not use the user’s personal data for ad targeting.

Third parties may process limited technical data in connection with ad display in accordance with their privacy policies.

6. Data Retention Period

  • image data is retained only for the time necessary for analysis,
  • scan history is stored locally on the user’s device,
  • no central database of user profiles is maintained.

7. User Rights

In accordance with the GDPR, the user has in particular the right:

  • to information about the processing of personal data,
  • to access their data,
  • to erasure of data (to the extent it is processed),
  • to object to processing,
  • to lodge a complaint with the supervisory authority.

8. Protection of Children

The Application is not intended for children under the age of 13 and does not target children as a specific group of users.

9. Changes to This Policy

The Controller reserves the right to update these privacy policies, especially in the event of changes to the Application’s functionality or legal regulations. The current version will always be publicly available.

10. Contact

If the user has any questions regarding privacy protection, they may contact the Controller via the email address provided above.
